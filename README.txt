Introduction
============

Did you came across a website in which you were really interested, and wanted
to check for any updates, or maybe you are intrested in getting updates if the
documentation of your favourite website was updated, Is the answer is yes,
Monitor-Web is your one stop solution, Monitor-Web tracks any changes in your
faviouite content and alerts you with proper log of differences. So, now never
waste time surfing the web to check if there are any updates, Simply add the website
to the website-list.txt and relax, Whenever you need to check simply run the monitor.py
and it will update the cache files and alert you if there are any changes. Monitor-Web
works best for static websites, mainly online HTML ebooks, online documentation, Udacity
cource lists ;) or something similar.

Dependencies
============

Monitor-Web is written in python and follows a precedural structure (which is expected to change soon)
It heavily lies on many different libraries most of them are standard and one of them are third party.

The only third-party library it supports is BeautifulSoup, developement site:-

    http://www.crummy.com/software/BeautifulSoup/

Usage
=====

To start Monitor-Web we have to set the proper environment which can be done using following.

[*] Install BeautifulSoup
[*] Download or Clone the repository
[*] and Run. ;)

Monitor-Web can be run using :-

    monitor.py
